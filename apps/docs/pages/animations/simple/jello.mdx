# Jello Animation

The **Jello** animation creates a jello-like effect by applying a series of skew transformations.

## Demo

<AnimationDemo type="jello">
  This element will jiggle like jello as it enters the viewport.
</AnimationDemo>

## Usage

To use the **Jello** animation, import it from `react-swift-reveal`:

```jsx filename="App.jsx" {3} copy
import { Jello } from 'react-swift-reveal';

const App = () => {
  return (
    <Jello>
      <div className="box">
        This element will jiggle like jello as it enters the viewport.
      </div>
    </Jello>
  );
};
```

## Props

Jello animation doesn't have any specific props of its own.
it has all the props of [Swift Reveal Props](/animations/props).

You can refer to the [Swift Reveal Props](/animations/props) documentation for a list of all available props.
