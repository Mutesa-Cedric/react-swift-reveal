# Spin Animation

The **Spin** animation creates a continuous rotation effect around the element's center.

## Demo

<AnimationDemo type="spin">
  This element will spin as it enters the viewport.
</AnimationDemo>

## Usage

To use the **Spin** animation, import it from `react-swift-reveal`:

```jsx filename="App.jsx" {3} copy
import { Spin } from 'react-swift-reveal';

const App = () => {
  return (
    <Spin>
      <div className="box">
        This element will spin as it enters the viewport.
      </div>
    </Spin>
  );
};
```

## Props

Spin animation doesn't have any specific props of its own.
it has all the props of [Swift Reveal Props](/animations/props).

You can refer to the [Swift Reveal Props](/animations/props) documentation for a list of all available props.
