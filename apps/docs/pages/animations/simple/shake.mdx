# Shake Animation

The **Shake** animation creates a rapid side-to-side shaking effect.

## Demo

<AnimationDemo type="shake">
  This element will shake as it enters the viewport.
</AnimationDemo>

## Usage

To use the **Shake** animation, import it from `react-swift-reveal`:

```jsx filename="App.jsx" {3} copy
import { Shake } from 'react-swift-reveal';

const App = () => {
  return (
    <Shake>
      <div className="box">
        This element will shake as it enters the viewport.
      </div>
    </Shake>
  );
};
```

## Props

Shake animation doesn't have any specific props of its own.
it has all the props of [Swift Reveal Props](/animations/props).

You can refer to the [Swift Reveal Props](/animations/props) documentation for a list of all available props.
