# Flash Animation

The **Flash** animation creates a quick flash effect by rapidly changing the opacity of an element.

## Demo

<AnimationDemo type="flash">
  This element will flash as it enters the viewport.
</AnimationDemo>

## Usage

To use the **Flash** animation, import it from `react-swift-reveal`:

```jsx filename="App.jsx" {3} copy
import { Flash } from 'react-swift-reveal';

const App = () => {
  return (
    <Flash>
      <div className="box">
        This element will flash as it enters the viewport.
      </div>
    </Flash>
  );
};
```

## Props

Flash animation doesn't have any specific props of its own.
it has all the props of [Swift Reveal Props](/animations/props).

You can refer to the [Swift Reveal Props](/animations/props) documentation for a list of all available props.
