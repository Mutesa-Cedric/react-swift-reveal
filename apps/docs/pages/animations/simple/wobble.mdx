# Wobble Animation

The **Wobble** animation adds a playful and wobbly effect to elements, creating a visually engaging transition.

## Usage

To use the **Wobble** animation, import it from `react-swift-reveal`:

```jsx
import { Wobble } from 'react-swift-reveal';

const App = () => {
  return (
    <Wobble>
      <div className="box">
        This element will wobble as it enters the viewport.
      </div>
    </Wobble>
  );
};
```

## Props

The **Wobble** animation doesn't have any specific props, but it accepts all the props that are available to all animations. You can read more about them in the [Swift Reveal Props documentation](/animations/props/).
