# Tada Animation

The **Tada** animation creates a celebratory effect by combining scaling and rotation.

## Demo

<AnimationDemo type="tada">
  This element will tada as it enters the viewport.
</AnimationDemo>

## Usage

To use the **Tada** animation, import it from `react-swift-reveal`:

```jsx filename="App.jsx" {3} copy
import { Tada } from 'react-swift-reveal';

const App = () => {
  return (
    <Tada>
      <div className="box">
        This element will tada as it enters the viewport.
      </div>
    </Tada>
  );
};
```

## Props

Tada animation doesn't have any specific props of its own.
it has all the props of [Swift Reveal Props](/animations/props).

You can refer to the [Swift Reveal Props](/animations/props) documentation for a list of all available props.
