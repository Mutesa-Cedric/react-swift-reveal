# Pulse Animation

The **Pulse** animation creates a subtle pulsing effect by scaling the element up and down.

## Demo

<AnimationDemo type="pulse">
  This element will pulse as it enters the viewport.
</AnimationDemo>

## Usage

To use the **Pulse** animation, import it from `react-swift-reveal`:

```jsx filename="App.jsx" {3} copy
import { Pulse } from 'react-swift-reveal';

const App = () => {
  return (
    <Pulse>
      <div className="box">
        This element will pulse as it enters the viewport.
      </div>
    </Pulse>
  );
};
```

## Props

Pulse animation doesn't have any specific props of its own.
it has all the props of [Swift Reveal Props](/animations/props).

You can refer to the [Swift Reveal Props](/animations/props) documentation for a list of all available props.
