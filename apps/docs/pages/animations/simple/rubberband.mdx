# RubberBand Animation

The **RubberBand** animation creates a rubber-like effect by stretching and squashing the element.

## Demo

<AnimationDemo type="rubberband">
  This element will stretch like a rubber band as it enters the viewport.
</AnimationDemo>

## Usage

To use the **RubberBand** animation, import it from `react-swift-reveal`:

```jsx filename="App.jsx" {3} copy
import { RubberBand } from 'react-swift-reveal';

const App = () => {
  return (
    <RubberBand>
      <div className="box">
        This element will stretch like a rubber band as it enters the viewport.
      </div>
    </RubberBand>
  );
};
```

## Props

RubberBand animation doesn't have any specific props of its own.
it has all the props of [Swift Reveal Props](/animations/props).

You can refer to the [Swift Reveal Props](/animations/props) documentation for a list of all available props.
