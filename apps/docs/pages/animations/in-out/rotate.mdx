# Rotate Animation

The **Rotate** animation creates a rotating entrance effect, making elements appear to spin in as they enter the viewport.

## Demo

<AnimationDemo type="rotate">
  This element will rotate in as it enters the viewport.
</AnimationDemo>

## Usage

To use the **Rotate** animation, import it from `react-swift-reveal`:

```jsx filename="App.jsx" {3} copy
import { Rotate } from 'react-swift-reveal';

const App = () => {
  return (
    <Rotate>
      <div className="box">
        This element will rotate in as it enters the viewport.
      </div>
    </Rotate>
  );
};
```

## Props

Rotate animation doesn't have any specific props of its own.
it has all the props of [Swift Reveal Props](/animations/props).

You can refer to the [Swift Reveal Props](/animations/props) documentation for a list of all available props.
