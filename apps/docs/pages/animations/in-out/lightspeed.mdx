# Lightspeed Animation

The **Lightspeed** animation creates a high-speed entrance effect, making elements appear to zoom in from the side with a blur effect.

## Demo

<AnimationDemo type="lightspeed">
  This element will zoom in at lightspeed as it enters the viewport.
</AnimationDemo>

## Usage

To use the **Lightspeed** animation, import it from `react-swift-reveal`:

```jsx filename="App.jsx" {3} copy
import { Lightspeed } from 'react-swift-reveal';

const App = () => {
  return (
    <Lightspeed>
      <div className="box">
        This element will zoom in at lightspeed as it enters the viewport.
      </div>
    </Lightspeed>
  );
};

export default App;
```

## Props

Lightspeed animation doesn't have any specific props of its own.
it has all the props of [Swift Reveal Props](/animations/props).

You can refer to the [Swift Reveal Props](/animations/props) documentation for a list of all available props.
