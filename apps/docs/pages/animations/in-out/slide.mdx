# Slide Animation

The **Slide** animation creates a smooth sliding entrance effect, making elements appear to slide in from the side.

## Demo

<AnimationDemo type="slide">
  This element will slide in as it enters the viewport.
</AnimationDemo>

## Usage

To use the **Slide** animation, import it from `react-swift-reveal`:

```jsx filename="App.jsx" {3} copy
import { Slide } from 'react-swift-reveal';

const App = () => {
  return (
    <Slide>
      <div className="box">
        This element will slide in as it enters the viewport.
      </div>
    </Slide>
  );
};
```

## Props

Slide animation doesn't have any specific props of its own.
it has all the props of [Swift Reveal Props](/animations/props).

You can refer to the [Swift Reveal Props](/animations/props) documentation for a list of all available props.
