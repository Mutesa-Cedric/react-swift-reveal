# Fade Animation

The **Fade** animation provides a smooth transition by gradually changing the opacity of an element.

## Demo

<AnimationDemo type="fade">
  This element will fade in as it enters the viewport.
</AnimationDemo>

## Usage

To use the **Fade** animation, import it from `react-swift-reveal`:

```jsx filename="App.jsx" {3} copy
import { Fade } from 'react-swift-reveal';

const App = () => {
  return (
    <Fade>
      <div className="box">
        This element will fade in as it enters the viewport.
      </div>
    </Fade>
  );
};
```

## Props

Fade animation doesn't have any specific props of its own.
it has all the props of [Swift Reveal Props](/animations/props).

You can refer to the [Swift Reveal Props](/animations/props) documentation for a list of all available props.
