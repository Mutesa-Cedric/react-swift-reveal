# Zoom Animation

The **Zoom** animation creates a smooth zooming entrance effect, making elements appear to scale up as they enter the viewport.

## Demo

<AnimationDemo type="zoom">
  This element will zoom in as it enters the viewport.
</AnimationDemo>

## Usage

To use the **Zoom** animation, import it from `react-swift-reveal`:

```jsx filename="App.jsx" {3} copy
import { Zoom } from 'react-swift-reveal';

const App = () => {
  return (
    <Zoom>
      <div className="box">
        This element will zoom in as it enters the viewport.
      </div>
    </Zoom>
  );
};
```

## Props

Zoom animation doesn't have any specific props of its own.
it has all the props of [Swift Reveal Props](/animations/props).

You can refer to the [Swift Reveal Props](/animations/props) documentation for a list of all available props.
