# Flip Animation

The **Flip** animation provides a 3D flip effect to elements, making them appear to flip in from the side.

## Demo

<AnimationDemo type="flip">
  This element will flip in as it enters the viewport.
</AnimationDemo>

## Usage

To use the **Flip** animation, import it from `react-swift-reveal`:

```jsx filename="App.jsx" {3} copy
import { Flip } from 'react-swift-reveal';

const App = () => {
  return (
    <Flip>
      <div className="box">
        This element will flip in as it enters the viewport.
      </div>
    </Flip>
  );
};
```

## Props

Flip animation doesn't have any specific props of its own.
it has all the props of [Swift Reveal Props](/animations/props).

You can refer to the [Swift Reveal Props](/animations/props) documentation for a list of all available props.
